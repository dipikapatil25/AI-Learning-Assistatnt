
# AI Learning Assistant Chatbot

## SDG 4: Quality Education

This project is an AI-powered educational assistant developed using Google ADK. It helps students with study tips, learning resources, exam preparation, and academic guidance.

## Features
- Study Tips
- Learning Resources
- Exam Preparation
- Time Management
- Motivation Support

## Technology Stack
- Google ADK
- Gemini 1.5 Flash
- Python
